### Banking Data Analytics

This is an end to end data analytics project which analyses a banking dataset and draws insights from it. 

Tech Stack used :
Python 
PowerBI
MySQL

Using python libaries including pandas ,numpy,seaborn and matplotlib I have analysed and interpreted the data. I have also visualized the data using heatmap and univariate and bivariate analysis. 

I have imported the dataset into mySQL WOrkbench and then connected to Python in vscode . 

Using PowerBI I have built 4 dashboards :HOME ,DEPOSIT ANALYSIS ,LOAN ANALYSIS AND SUMMARY.
